# 🚗 Car Price Prediction Using Machine Learning

## 📌 Project Overview

This project predicts car prices using Machine Learning techniques. The model analyzes various car features such as company, year, fuel type, kilometers driven, and transmission type to estimate the price of a car.

## 🎯 Objective

The main objective of this project is to build a machine learning model that can predict the price of a car accurately based on different input features.

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook / Google Colab

## 📂 Dataset

The dataset contains information about:

* Car Company
* Car Model
* Year
* Fuel Type
* Kilometers Driven
* Price

1. Data Collection

4. Feature Selection
5. Model Training
6. Prediction
7. Model Evaluation
